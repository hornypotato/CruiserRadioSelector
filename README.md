# Cruiser Radio Selector

A small quality-of-life mod that lets you pick cruiser radio tracks directly with a ingame menu.

No more spamming the "next track" button 30 times — just select what you want instantly.

---

## Features

![](https://files.catbox.moe/eq32f5.png)

- Select any track directly
- Supports large track lists (with pages)
- Simple in-game menu

---

## Controls

- **Hold C** — open track selector
- **Hold C + 1–9 / 0** — select track
- **Hold C + V** — switch page

---
